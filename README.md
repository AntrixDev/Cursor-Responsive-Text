## Demo
Move your mouse left and right to see the effect in action.

## Features
- Split text layout
- Smooth CSS transitions

## How It Works
- The text is duplicated into two containers.
- Each side is moving in opposite directions using CSS transforms.
- JavaScript listens to mouse movement and translates the text based on cursor position.

## Getting Started
1. Clone the repository
2. Open index.html in your browser
3. Move your mouse and enjoy:]
